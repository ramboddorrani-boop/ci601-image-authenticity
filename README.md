# CI601 — Image Authenticity Classifier

**Rambod Dorrani · Student ID 22843936 · University of Brighton**
Supervisor: Sanaz Fallahkhair · Second Reader: Shine Shan

A binary classifier that distinguishes AI-generated images from real
photographs, deployed as a Flask web application. The model is a
ResNet50 (ImageNet-pretrained, backbone frozen) with the final
classification layer retrained on a custom 24,484-image dataset built
from Midjourney v6.1, Nano Banana, Unsplash, and FFHQ.

Three forensic signals (Error Level Analysis, EXIF metadata inspection,
Laplacian noise variance) run alongside the classifier and are shown
on the result page as independent supporting evidence.

Test accuracy: **98.7%** on a balanced held-out set of 3,674 images
(F1 = 0.987 on both classes). The full methodology, results, and
critical review are in the accompanying dissertation report.

## Quick start

Tested with Python 3.10+.

```bash
# 1. Clone this repo
git clone <repo-url>
cd <repo>

# 2. Download best.pt from the Google Drive link above
#    and place it at: models/best.pt

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run the app
python app.py
```

Open http://localhost:8000 and drop in an image. The verdict,
confidence score, and three forensic signals appear on a single
result page.

The app auto-selects an inference device: Apple Silicon MPS if
available, then CUDA, then CPU. CPU inference takes 1–2 seconds
per image.

## Reproducing the results

Training was done on a Kaggle notebook with dual T4 GPU
acceleration. The full pipeline takes about 47 minutes.

1. Upload the three source datasets to Kaggle:
   - The AI image dataset (Midjourney v6.1 + Nano Banana)
   - Unsplash images (`jettchentt/unsplash-dataset-images-downloaded-250x250`)
   - FFHQ face subset (`tommykamaz/faces-dataset-small`)
2. Run `notebooks/kaggle_train.py` as a Kaggle notebook
3. Download `best.pt` from `/kaggle/working/`

Hyperparameters: ResNet50, frozen backbone, Adam optimiser,
cross-entropy loss, batch size 64, max 15 epochs, early stopping
patience 3, seed 42. Training stopped at epoch 13 with the best
weights from epoch 10.

To regenerate the report figures from a saved evaluation run:

```bash
python src/make_figures.py
```

## Method summary

- **Model:** ResNet50 (ImageNet-pretrained, backbone frozen). Only
  the final fully-connected layer is replaced with a `Linear(2048, 2)`
  and trained.
- **Dataset:** 12,242 AI images (Midjourney v6.1 + Nano Banana, all
  generated by the author at Dreamcatcher Films) + 9,181 Unsplash
  photographs + 3,061 FFHQ faces. Split 70/15/15 with seed 42.
- **Forensics:** ELA re-encodes the image at JPEG quality 90 and
  reports the standard deviation of the pixel-level difference. EXIF
  parses the JPEG header for camera tags. Noise variance applies a
  pure-NumPy Laplacian convolution and reports its standard deviation.
  Thresholds were calibrated on 100 validation images per class.

## Limitations

The 98.7% test accuracy is **in-distribution**: the test images come
from the same generators (Midjourney v6.1, Nano Banana) that supplied
the training data. Real-world performance against generators the
classifier has never seen — Stable Diffusion 3, DALL-E 3, FLUX,
Imagen 3 — is unknown and may be substantially lower (Corvi et al.,
2023). The forensic signals partly mitigate this by providing
independent evidence the user can interrogate, but they do not close
the cross-generator generalisation gap.

Full discussion in §5 (Critical Review) of the dissertation report.
